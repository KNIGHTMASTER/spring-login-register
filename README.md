# spring-login-register
An example Spring Boot Application for Login through web services.

# Technologies !
WEB :
  - Spring Web
  - Spring Security
  - Thymeleaf
  - Bootstrap
  - Jquery

API :
  - Spring Web
  - JPA
  - MySql
